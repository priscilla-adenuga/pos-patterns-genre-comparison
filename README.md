# POS Patterns Across Genres (News vs Tweets)

This project compares part-of-speech (POS) patterns across two text genres using spaCy:
- News-style text
- Tweet-style text

## Goals
- Run POS tagging with spaCy
- Compare grammatical patterns across genres
- Interpret differences from a linguistics perspective

## Methods
For each genre, we compute:
- POS distribution (NOUN, VERB, ADJ, PRON, etc.)
- Noun–Verb ratio
- Most frequent POS bigrams (e.g., ADJ + NOUN)

## Structure
- data/ → text samples
- notebooks/ → analysis notebook
